# Magento2 Integration Bot Docs

This folder contains concise reference docs for how the service is structured, how requests flow, and how to extend it.

## Contents
- `HLD.md` High-level design, responsibilities, and system boundaries.
- `LLD.md` Low-level design details by package and class responsibilities.
- `FLOW.md` Request/response flow for key use cases.
- `ARCHITECTURE.md` Layered architecture and dependency direction.
- `EXTENDING.md` How to add new Magento endpoints in the same style.
